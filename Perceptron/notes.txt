en utilisant un hyperplan de l'espace séparateur.

Entrée :        S = (xi, yi) 1<=i<=m
                Un # max d'itérations : T
                Un pas d'apprentissage : mu
Initialisation :        w0(1) <- 0
                        Rd inclus w(1) <- o->

for t = 1..T
        for i = 1..m
                if( yi(w0(t) + <w(t),xi>) <= 0 )
                        w0(t+1) <- w0(t) + mu.yi
                        w(t+1) <- w(t) + mu.xi.yi
